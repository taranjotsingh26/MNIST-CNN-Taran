# MNIST Digit Recognition using CNN

This project uses TensorFlow and CNN to classify handwritten digits from the MNIST dataset.

## 🔧 Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## 📊 Model
- Convolutional Neural Network (CNN)
- Trained on MNIST dataset

## 🚀 How to Run
Open the notebook in Google Colab and run all cells.

## 📁 File
- Experiment_4_CNN_MNIST.ipynb
